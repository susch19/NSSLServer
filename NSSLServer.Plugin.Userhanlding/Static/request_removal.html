Deutsch:

Löschung der Benutzerdaten von NSSL

Sämtliche Daten, bis auf das Konto selbst, können innerhalb der NSSL App gelöscht werden.
Sobald eine Liste über die Seitenliste aus der App entfernt wird, sind alle damit verbundenen Daten entgültig gelöscht.
Um das Konto zu löschen, muss eine E-Mail an den Entwickler nssl@outlook.com gesendet werden.
In der E-Mail reicht es einfach zu sagen, dass eine Kontolöschung beantragt wird.
Die E-Mail muss von dem Konto gesendet werden, das zu dem Konto gehört.
Innerhalb von 30 Tage wird eine Antwort gesendet, wo eine zusätzliche Bestätigung benötigt wird, um Missbrauch zu verhinden.

Sollen anderen Daten gelöscht oder geändert werden, kann dies unter der gleichen E-Mail Adresse passieren.
Hierfür sollte beschrieben werden, was gelöscht / geändert werden soll.
Bei Rückfragen meldet sich der Entwickler.


English:

Deletion of Userdata for NSSL

All data, except for the account itself, can be deleted within the NSSL app.
Once a list is removed from the app via the page list, all associated data is permanently deleted.
To delete the account, an email must be sent to the developer at nssl@outlook.com.
In the email, it is sufficient to simply state that a request for account deletion is being made.
The email must be sent from the account that belongs to the account.
A response will be sent within 30 days, where additional confirmation is required to prevent abuse.

If other data needs to be deleted or changed, this can be done at the same email address.
For this, it should be described what should be deleted/changed.
If there are any questions, the developer will get in touch.
